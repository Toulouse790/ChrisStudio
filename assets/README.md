# ChrisStudio Assets Library

## Structure des dossiers

```
assets/
├── music/                    # Musiques de fond (génère avec Suno)
│   ├── epic/                 # Pour "What If..." - épique, cinématique
│   ├── mysterious/           # Pour "Classified Files" - sombre, suspense
│   ├── orchestral/           # Pour "Human Odyssey" - classique, majestueux
│   ├── ambient/              # Ambiances génériques
│   └── transitions/          # Sons de transition (whoosh, impact)
│
├── videos/                   # Clips vidéo (6-8 secondes idéalement)
│   ├── what-if/              # Futuriste, sci-fi, technologie
│   ├── human-odyssey/        # Historique, archival, civilisations
│   ├── classified-files/     # Mystérieux, sombre, documents
│   └── generic/              # Clips polyvalents
│
├── images/                   # Images haute qualité
│   ├── what-if/              # Futuriste, conceptuel
│   ├── human-odyssey/        # Historique, vintage
│   ├── classified-files/     # Mystérieux, noir
│   └── generic/              # Images polyvalentes
│
└── sfx/                      # Effets sonores
    # Exemples de noms:
    # - whoosh.mp3
    # - reveal.mp3
    # - tension.mp3
    # - impact.mp3
    # - transition_dark.mp3
    # - drone_ambient.mp3
```

## Recommandations

### Musique (Suno)
- **Durée** : 3-5 minutes par track
- **Quantité** : 5-10 tracks par mood
- **Format** : MP3 ou WAV
- **Prompts Suno suggérés** :
  - Epic : "cinematic orchestral epic trailer music, no vocals, building tension"
  - Mysterious : "dark ambient mystery documentary music, suspenseful, no vocals"
  - Orchestral : "classical documentary music, historical, majestic, no vocals"

### Vidéos
- **Durée** : 6-8 secondes idéalement
- **Résolution** : 1080p minimum (1920x1080)
- **Format** : MP4 (H.264)
- **FPS** : 24-30 fps

### Images
- **Résolution** : 1920x1080 minimum
- **Format** : JPG ou PNG
- **Ratio** : 16:9 préféré

### Effets Sonores (SFX)
- **Durée** : 0.5-3 secondes
- **Format** : MP3 ou WAV
- **Types utiles** :
  - Whoosh (transitions)
  - Impact (révélations)
  - Tension risers
  - Ambient drones
